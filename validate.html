pipeline.validate.compare_data(vname: str, xbox, kerchunk_box, logger, bypass=False)

Compare a NetCDF-derived ND array to a Kerchunk-derived one - Takes a netcdf selection box array of n-dimensions and an equally sized kerchunk_box array - Tests for elementwise equality within selection. - If possible, tests max/mean/min calculations for the selection to ensure cached values are the same.

  • Expect TypeErrors from summations which are bypassed.

  • Other errors will exit the run.
